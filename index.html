Java

¿Cuál es su Historia?

Java es un lenguaje de programación que fue desarrollado por Sun Microsystems. Java fue presentado en la segunda mitad del año 1995 y desde entonces se ha convertido en un lenguaje de programación muy popular. Java está muy bien valorado porque los programas Java se pueden ejecutar en distintas plataformas con sistemas operativos como Windows, MacOS, Linux y Solaris. James Gosling, el director del equipo de trabajo encargado de desarrollar Java, hizo real la promesa de un lenguaje independiente de la plataforma. La intención era crear un lenguaje que permitiera programar una aplicación una sola vez que luego pudiera ejecutarse en distintas máquinas y sistemas operativos. Para conseguir la portabilidad de los programas Java se utiliza un entorno de ejecución para los programas compilados. Este entorno recibe el nombre de Java Runtime Environment (JRE). Es gratuito y está disponible para los principales sistemas operativos (Windows, MasOS, Linux y Solaris).

Plataforma de Java

Los programas Java se compilan a un lenguaje intermedio, denominado Bytecode. Este código es interpretado por la máquina virtual de Java del entorno de ejecución (JRE) y se consigue la portabilidad en distintas plataformas. Estructura de un programa Java

En Java se identifican diferentes elementos: comentarios, definiciones de clase, definiciones de método y sentencias.

  • Comentario: El delimitador de inicio de un comentario es /* y el delimitador de fin de comentario es */. Los comentarios son ignorados por el compilador pero son útiles para el programador. Los comentarios ayudan a explicar aspectos relevantes de un programa y lo hacen más legible.
  • Definición de clase:La definición de una clase comienza por el carácter { y termina con el carácter }. El nombre de la clase sera determinado por el programador.
  • Definición de método: Una vez escrita la definición de clase se escribe la definición del método main(). Todos los programas Java deben incluir un método main(). Este método es una secuencia de sentencias ejecutables. Las sentencias de un método quedan delimitadas por los carácteres { y } indicando el inicio y final del método.
  • Sentencia: Dentro del método main() se pueden incluir sentencias que tras ser compiladas apareceran en la consola, para que la orden aparezca la sentencia deberá llevar punto y coma para indicar al compilador que ha finalizado la sentencia.

Variables

Las variables de Java pueden ser de un tipo primitivo de datos o una referencia a un objeto. Los tipos primitivos permiten representar valores básicos. Estos tipos se clasifican en números enteros, números reales, caracteres y valores booleanos.

  • Números enteros: representan números enteros positivos y negativos con diferentes rangos de valores. Los tipos enteros en Java son byte, int, short y long
  • Números reales: Existen dos tipos de números reales en Java, float y double. La diferencia entre ellos está en el número de decimales que tienen capacidad para expresar y en sus rangos de valores
  • Caracteres: El tipo char permite representar cualquier carácte Unicode. Los caracteres Unicode contienen todos los caracteres del alfabeto de la lengua castellana.
  • Booleano: Se utiliza para la representacion de valores lógicos verdadero y falso. (Dos valores true y false.

    • Operadores

    Cada tipo puede utilizar determinados operadores para realizar operaciones o cálculos.

    • Números enteros: Con los números enteros se pueden realizar operaciones unarias, aditivas, multiplicativas, de incremento y decremento, relacionales, de igualdad y de asignación.
      1. Una operación unaria permite poner un signo delante: +5, -2.
      2. Una operación aditiva es simplemente una suma o resta: 2+3, 5-2.
      3. Una operación multiplicativa multiplica o divide valores: 5*2, 5/2. El operador % calcula el residuo de la división: 5%2.
      4. Un incremento o decremento aumenta o decrementa en 1 el valor de una variable: ++num, num++, --num, num--. Si el operador va después de la variable su valor se modifica al final, en cambio si va delante se realiza la operación y se modifica el valor de la variable.
      5. Un operador relacional permite comparar dos valores: >, <, >= y <=.
      6. Un operador de igualdad compara si dos valores son iguales. El operador == devuelve verdadero si los dos valores son iguales, el operador != devuelve verdadero si son diferentes.
      7. Un operador de asignación permite asignar un valor o el resultado de una operación a una variable: =, +=, -=, *=, /=, %=.
    • Números reales: como se ha dicho antes los números reales utilizan los mismos operadores que los números enteros.
    • Booleanos: Los operadores que se aplican a los valores lógicos son: negación, Y lógico, O lógico.
      1. La negación (!) devuelve true si el operador es false.
      2. El Y lógico (&&) devuelve false si uno de los operadores es false.
      3. El O lógico (||) devuelve true si uno de los operadores es true.

    Clases y objetos

    La programación orientada a objetos se enfoca en los elementos de un sistema, los atributos y las interacciónes que se producen entre ellos para el diseño de aplicaciones informáticas. Estos elementos reciben el nombre de clases.
    Las clases representan un esquema simplificado de la casuítica de una aplicación informática. Es una representación abstracta de un conjunto de objetos los cuales comparten los mismos atributos y el mismo comportamiento.

    Clases

    En Java una clase se define por la palabra reservada class seguida del nombre de la clase. (El nombre de la clase debe comenzar por mayúscula). Cuando se declara una clase se pueden crear objetos a partir de ella, es necesario definir la información que se almacena el objeto y su comportamiento.

    Atributos

    Los atributos pueden ser de tipos primitivos de Java o de tipo objeto. Pongamos un ejemplo: Tenemos un catálogo de coches de una empresa, para elegir nuestro coche necesitamos conocer la matrícula del coche, su marca, el modelo, el color, el precio y si este está disponible.

    Métodos y constructores

    A parte de definir los atributos también hay que definir los métodos que determinan su comportamiento. Toda clase debe definir un método especial. Este método tiene el mismo nombre que la clase, por ejemplo para la clase Coche , el identificador del método también se llamará así. El método constructor se ejecuta cada vez que se instancia un objeto de la clase. Este método se usa para la inicialización de los atributos del objeto que se instancia.
    Para diferenciar entre los atributos y los identificadores de los parámetros de método constructor se utiliza la palabra this. De esta forma los parámetros del método pueden tener el mismo nombre que la clase. Esto nos permitirá hacer una asignación. Por ejemplo podemos escribir this.marca refiriendonos al atributo del objeto y marca para el parámetro del método.
